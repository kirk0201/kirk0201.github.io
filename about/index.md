gulp로 css minimalize
`gulp css`

코드 하이라이트 rouge 설치 및 사용방법

rouge를 설치한다
`gem install rouge`

하이라이트 목록을 받아온다
`rougify help style`

syntax.css로 덮어쓰면 default설정을 추가하지 않아도 적용됨
`rougify style (style이름) > assets/css/syntax.css`

스타일을 바꾸면 gulp css를 해주자자
